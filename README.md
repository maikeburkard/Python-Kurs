# Python-Kurs


## Abenteuerspiel

### 1. Wald oder Stadt?

Das Programm fragt: "Möchtest du in den Wald oder in die Stadt gehen?"

Wenn der Mensch "Wald" eingibt, antwortet das Programm mit "Du gehst in den Wald".

Wenn der Mensch "Stadt" eingibt, antwortet das Programm mit "Du gehst in die Stadt".

Sonst sagt das Programm: "Bitte gib Wald oder Stadt ein"
